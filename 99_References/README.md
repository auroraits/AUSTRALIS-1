# Reference material policy

Do not vendor third-party PDFs, standards, brochures, datasheets, papers or
auto-generated transcripts in the public repository.

Use links, citations and project-authored summaries instead. If a reference must
be stored internally for private work, keep it outside the public mirror or in a
private artifact store with clear provenance and redistribution rights.

The previous public-tree audit removed copies of:

- CubeSat Design Specification PDFs;
- NASA CubeSat 101 PDF;
- Iridium brochure;
- academic papers and abstracts;
- vendor/customer information documents;
- auto-generated video transcript.
